# Manual técnico HET-CD

Este repositorio incluye el manual técnico completo en formato Word:

- `manual_tecnico_het_cd.docx`

Y una guía rápida de operación:

- `guia_rapida_het_cd.docx`

Ambos documentos explican el modelo de valoración, la estructura del archivo de datos, los criterios CD1.1-CD6.6, el uso paso a paso de la app, la interpretación de resultados, el impacto económico, el arrastre y el despliegue en Streamlit.
